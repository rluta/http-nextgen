HTTP/2 Support


Most advanced server are currently h2o and nghttp2

 

Firefox 36 has h2 enabled. Chrome 40+ has h2-14.

 

Nghttpx can be used as a h2 frontend proxy to your server 


        $ nghttpx -f 0,443 -b localhost,8080

 

Applications can easily support push with Link response header 


        Link: </fonts/myfont.woff>; rel=preload
