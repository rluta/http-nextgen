HTTP/2 secure proxy

  • HTTP/2 intermediary can be used as a secure proxy for outgoing requests
  • Provides network optimization between client and proxy
  • Provides confidentiality
  • Does not require a local agent installed

To try it out:


docker run -it -p 3000:3000 -p 443:443 rluta/h2proxy run.sh proxy h2Proxy.mydomain.com
  • Configure DNS or host entry
  • Accept self-signed certificate by connecting to https://h2Proxy.mydomain.com/
  • Configure browser to use PAC file https://h2proxy.mydomain.com/secure.pac
