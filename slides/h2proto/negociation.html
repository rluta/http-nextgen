HTTP/2 Negociation

ALPN - h2 protocol

TLS extension for negociating application protocol during handshake

Dependant on TLS so communication are always encrypted

Preferred and recommended way to deploy HTTP/2

HTTP Upgrade - h2c protocol

Use HTTP/1.1 Upgrade mechanism, similar to WebSocket

May be proxy unfriendly but does not require encryption


    GET / HTTP/1.1
    Host: www.work4labs.com
    Upgrade: h2c
    Connection: Upgrade, HTTP2-Settings
    HTTP2-Settings: <base64 SETTINGS>

    HTTP/1.1 101 Switching protocol
    Upgrade: h2c
    Connection: Upgrade

    <HTTP2 frame data>
