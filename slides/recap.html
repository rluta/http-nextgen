Recap

SSE and Websockets

  • Reactive, realtime applications
  • Simple and powerful Javascript API
  • Need backend work for large scale deployments

SPDY and HTTP/2

  • Faster web delivery, especially for mobile
  • Infrastructure can easily be upgraded...
  • ...if you already support TLS
  • Need work on dev side for maximum benefits
  • Server support still immature

Expect convergence between WebSockets and HTTP/2 to happen "soon"

Compatibility with proxy/cache is still a work in progress (Trusted Proxies)
